Descripción del proyecto

Diseñé la identidad visual para un concierto benéfico que recaudó fondos para cubrir los costos clínicos por problemas de salud que ha enfrentado el artista y compositor chileno, José Arturo Chávez. Este proyecto incluyó la creación de un logotipo distintivo, así como una serie de piezas promocionales elaboradas para medios digitales e impresos.
